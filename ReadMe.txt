문제 풀이를 완료했씁니다.

해당 문제는 1, 2, 3이 첫 번째 자리에 위치할 경우 그 이후 값의 경우의 수를 생각해 풀면 간단하게 풀 수 있습니다.

현재 1의 경우의 수는 1, 2의 경우의 수는 2, 3의 경우의 수는 4입니다.

여기서 4의 경우의 수를 구할 경우, 1, 2, 3이 첫 번째 자리로 나오는 경우의 수를 구해줍니다.

1이 첫 번째 자리에 나오는 경우 4에서 1을 뺀 나머지, 3의 경우의 수가 해당 경우의 수가 됩니다.

2가 첫 번째 자리에 나오는 경우 4에서 2를 뺀 나머지, 2의 경우의 수가 해당 경우의 수가 됩니다.

3이 첫 번째 자리에 나오는 경우 4에서 3을 뺀 나머지, 1의 경우의 수가 해당 경우의 수가 됩니다.

그러므로 4의 경우의 수는 4 + 2 + 1 = 7이 됩니다.

이와 같이 나머지 수들도 경우의 수를 구할 수 있습니다.
